# InkFlow

A real-time collaborative whiteboard that enables multiple users to draw, brainstorm, and share ideas seamlessly through live synchronization.

## Features

- Real-time collaboration
- Shared interactive canvas
- Room-based sessions
- WebSocket-powered synchronization
- Color and brush customization
- Responsive design

## Tech Stack

**Frontend:** React, Vite, Canvas API  
**Backend:** FastAPI, WebSockets  
**Deployment:** Vercel

## Roadmap

- Undo/Redo
- Shape and text tools
- Board persistence
- Export functionality
- Live user cursors
- AI-powered diagram generation
